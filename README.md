## üìò What is this repository?

This repository contains **Jupyter notebooks that accompany the open educational resource (OER) textbook** for *Physics in Medicine & Biology*.

The notebooks are designed to:
- Reinforce key concepts from the textbook
- Provide **interactive, visual explanations** of physical principles in medicine
- Allow students to explore models by **changing parameters and re-running cells**
- Support students with **no prior programming experience**

You do **not** need to be a programmer to use these notebooks.

## üß≠ How this repository is organized

- Each notebook corresponds to a **chapter or subsection** of the textbook  
- Notebooks are scaffolded from **very beginner-friendly** to more advanced (optional) explorations  
- Advanced code is always **pre-filled** ‚Äî you are never expected to write complex code from scratch
